# PVPSIT Mate 🎓

An AI assistant that answers questions about PVPSIT using official college documents.

## Features
- Ask questions about college information
- Answers generated from official PDFs
- Built using RAG (Retrieval Augmented Generation)

## Tech Stack
- Streamlit
- LangChain
- FAISS
- HuggingFace Embeddings
- Groq LLM
